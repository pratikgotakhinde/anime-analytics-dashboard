# Anime Analytics Dashboard

Interactive dashboard analyzing 108 years of anime data spanning 1917 to 2025.

## Features

- Overview with key metrics and distributions
- Hidden gems analysis identifying underrated anime
- Genre trends and evolution over decades  
- Interactive search and filtering

## Tech Stack

- Python 3.10+
- Streamlit for the dashboard
- Plotly for visualizations
- Pandas for data processing
